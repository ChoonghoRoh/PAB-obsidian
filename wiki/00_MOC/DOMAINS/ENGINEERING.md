# ENGINEERING — Domain MOC

## DOMAIN 정의

`ENGINEERING`은 **일반 소프트웨어 공학 도메인**이다. AI/HARNESS에 종속되지 않는 범용 SW 지식 — 알고리즘·자료구조·프로그래밍 언어·시스템 디자인·디버깅·성능 — 의 노트를 모은다.

**포함 주제 예시**: Python, TypeScript, system design, algorithm, data structure, design pattern, profiling, distributed systems, SQL, git workflow
