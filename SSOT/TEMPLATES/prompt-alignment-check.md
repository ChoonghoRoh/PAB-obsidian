### diff 유형 분류

| 유형 | 의미 |
|------|------|
| **신규 추가** | 원본 프롬프트 기능이 새 코드로 구현 |
| **수정** | 기존 코드를 변경하여 반영 |
| **삭제** | 기존 코드가 프롬프트 요구에 의해 제거 |
| **미변경** | 기존 코드가 이미 프롬프트 충족 |
